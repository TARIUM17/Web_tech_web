# Web_tech_web
***Тема сайта* - Сайт робототехнической компании CyberFun Tech.**
    Данное приложение будет представлять основной и официальный сайт робототехнической компании. На нём будетпредставлена информация о компании, её структре, партнёрах и разработках. Также будет присутствовать возможность оставить заявку на рассмотр резюме.

## *Целевая аудитория:* потенциальные инвесторы, партнёры, выпускники робототехнических коледжей и университетов, а также простые люди.

- Главная
- Аутентификация
- Каталог разработок
- Личная страница пользователя
- Перечень отмеченных товаров "Избранное"
- Подача заявкина рассмотрение разработки
- Интерактивная страница с миии-игрой для юных посетителей
- Админ-панель

**Технологический стек**
- Frontend: чистый JS, роутер, state-менеджмент.
- Backend: язык Node.Js, БД - Supabase.

## ***Визуал***

### - Главная
Тип доступа: Публичный
- Шапка (лого, слоган, меню)
- Проигрывание интерактивной анимации и попытка запроса данных
- Основная информация о компании и приветствие для потенциальных новых сотрудников
- Отображение блока, разбитого на несколь сегментов. В какждом из них краткая информация о сотруднике
- Футер с контактами

### - Аутентификация
Тип доступа: Приватный
- Шапка (лого, слоган, меню)
- Поля для ввода информации (Различаются в зависимости от выбранного режима)
- Футер с контактами

### - Каталог разработок
Тип доступа: Публичный
- Шапка (лого, слоган, меню)
- Горизонтальный каталог с изображениями и информацией о конкретной разработке
- Футер с контактами

### - Личная страница пользователя
Тип доступа: Приватный
- Шапка (лого, слоган, меню)
- Ссылки на партнёров (рекомендации)
- Ссылка на перечень закрепений
- Отправка формы с разработкой для компании
- Футер с контактами

### - Перечень отмеченных товаров избранное
Тип доступа: Приватный
- Шапка (лого, слоган, меню)
- Список закреплённых товаров
- Футер с контактами

### - Подача заявкина рассмотрение разработки
Тип доступа: Приватный
- Шапка (лого, слоган, меню)
- Поле для внесения схем, информации и преимуществ о данной разработке
- Футер с контактами

### - Мини игра
- Шапка (лого, слоган, меню)
- Поле с небольшой локацией и изображением мальникого робота
- Поле со списком действий и кнопками для создания алгоритма движения робота

### - Админ-панель
Тип доступа: Приватный
- Шапка (лого, слоган, меню)
- просмотр пользовательских записей и управление
- Редактирование инфорации об определённом проекте
- Исзменение оценки "значимости" сотрудника для компании
- Футер с контактами

### User (Пользователь)

| Поле          | Описание                              |
| ------------- | ------------------------------------- |
| id            | Уникальный идентификатор пользователя |
| email         | Адрес почты, используется для входа   |
| username      | Отображаемое имя пользователя         |
| password_hash | Захешированный пароль                 |
| secret word   | Секретное слово для восстановления    |
| created_at    | Дата и время регистрации              |

### Developments

| Поле        | Описание                                        |
| ----------- | ----------------------------------------------- |
| id          | Уникальный идентификатор проекта                |
| name        | Название проекта (3-100 символов)               |
| description | Краткое описание проекта                        |
| img         | Цена проекта                                    |
| new         | Пометка о корректности разработки с возможностю |
|             | её отзыва                                       |

### Engineers

| Поле        | Описание                                     |
| ----------- | -------------------------------------------- |
| id          | Уникальный id сотрудника                     |
| name        | Имя сотрудника                               |
| surname     | Фамилия сотрудника                           |
| pic         | Портрет с карточки сотрудника                |
| important   | Специальная отметка об успехах сотрудника    |
| dev         | Перечень разработок и проектов сотрудника    |

## Ключевые эндпоинты API

### *Авторизация*
| Метод | Эндпоинт                | Описание                         |
| ----- | ----------------------- | -------------------------------- |
| POST  | `/api/auth/register`    | Регистрация нового пользователя  |
| POST  | `/api/auth/login`       | Вход в аккаунт по email и паролю |
| POST  | `/api/auth/logout`      | Выход из аккаунта                |
| GET   | `/api/auth/check-email` | Проверка уникальности email      |

### *Разработки*
| Метод  | Эндпоинт              | Описание                                     |
| ------ | --------------------- | -------------------------------------------- |
| GET    | `/api/products`       | Получить список разработок с фильтрацией     |
| GET    | `/api/products/:id`   | Получить id информацию о разработке          |
| POST   | `/api/products`       | Внести новое решение                         |
| PATCH  | `/api/products/:id`   | Обновить данные решения                      |
| DELETE | `/api/products/:id`   | Удалить информаци о решениии                 |
